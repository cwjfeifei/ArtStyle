# 艺术风格展示系统

一个交互式的艺术风格浏览和提示词生成工具，支持 100+ 种艺术风格的可视化展示和 CSS 滤镜模拟。

## 功能特性

### 核心功能
- **艺术风格库**：内置 100+ 种艺术风格，涵盖古典、现代、抽象、数字艺术等多个流派
- **实时搜索**：支持按风格名称或标签快速搜索和筛选
- **视觉模拟**：通过 CSS 滤镜和纹理叠加模拟不同艺术风格的视觉效果
- **随机配图**：使用 Picsum 提供的随机图片展示风格效果
- **提示词生成**：一键复制风格描述和标签，方便用于 AI 绘画工具

### 风格分类
- **古典艺术**：霍布马、米勒、德加、莫奈、伦勃朗等大师风格
- **现代艺术**：毕加索立体派、达利超现实主义、波洛克抽象表现主义
- **数字艺术**：赛博朋克、蒸汽朋克、合成波、像素艺术、故障艺术
- **动画风格**：新海诚、宫崎骏、浮世绘
- **特殊效果**：水彩、素描、双重曝光、玻璃拟态、酸性设计

### 视觉效果模拟
系统根据风格标签自动应用对应的视觉效果：
- **赛博朋克**：霓虹色调、高饱和度、扫描线纹理
- **蒸汽朋克**：复古棕色、噪点纹理、怀旧滤镜
- **像素艺术**：像素化渲染、网格纹理
- **水彩画**：柔和模糊、低对比度
- **故障艺术**：色彩失真、画面撕裂效果
- **极简主义**：去饱和、高亮度、大量留白

## 使用方法

### 浏览风格
1. 在左侧列表中浏览所有艺术风格
2. 点击任意风格名称查看详细信息和视觉效果
3. 使用顶部搜索框快速查找特定风格

### 复制提示词
1. 选择喜欢的艺术风格
2. 点击"复制提示词"按钮
3. 提示词格式：`风格名称，标签1、标签2、标签3...`
4. 可直接用于 Midjourney、Stable Diffusion 等 AI 绘画工具

### 切换示例图
- 点击"切换随机图"按钮查看该风格在不同图片上的效果
- 每次切换会加载新的随机图片并应用当前风格滤镜

## 技术实现

### 前端技术
- **纯 HTML/CSS/JavaScript**：无需任何框架或依赖
- **CSS 变量**：动态主题色系统
- **Canvas 动画**：背景粒子效果
- **CSS 滤镜**：实时视觉效果模拟
- **响应式设计**：支持桌面和移动设备

### 核心特性
- **模块化架构**：ArtEngine 类封装所有逻辑
- **配方系统**：预定义的风格配置模板
- **纹理系统**：SVG 纹理叠加（噪点、扫描线、网格、半色调）
- **智能匹配**：根据标签关键词自动应用对应效果

### 性能优化
- 图片预加载机制
- 平滑过渡动画
- 高效的 Canvas 粒子系统
- 最小化 DOM 操作

## 扩展开发

### 添加新风格
在 `rawData` 字符串中按以下格式添加：
```
风格名称、标签1、标签2、标签3、标签4。
```

### 自定义视觉效果
在 `recipes` 对象中添加新配方：
```javascript
newStyle: {
    keys: ['关键词1', '关键词2'],
    theme: '#颜色代码',
    filters: 'CSS滤镜效果',
    texture: '纹理样式',
    border: '边框样式'
}
```
